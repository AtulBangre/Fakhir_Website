# Fakhri IT Services

Official website for Fakhri IT Services, a premier agency dedicated to empowering Amazon sellers with expert account management, advertising strategies, and growth solutions. This modern, responsive web application is built using Next.js and Tailwind CSS.

## 🚀 Features

-   **Modern UI/UX**: Premium, responsive design with smooth animations and transitions using Framer Motion.
-   **Service Showcases**: Detailed pages for various Amazon services including Account Management, PPC, and Listing Optimization.
-   **Priority Support**: Dedicated "Within 2 Hours" service section for urgent task handling.
-   **Dynamic Blog**: Feature-rich blog system with category filtering and social sharing capabilities.
-   **Pricing Plans**: Clear, interactive pricing tables for different service tiers.
-   **Contact Integration**: Professional contact forms and location details.
-   **SEO Optimized**: Built-in SEO best practices for better visibility.

## 🛠️ Tech Stack

-   **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
-   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
-   **Animations**: [Framer Motion](https://www.framer.com/motion/) & [ScrollReveal](https://scrollrevealjs.org/)
-   **Icons**: [Lucide React](https://lucide.dev/)
-   **Scrolling**: [Lenis](https://github.com/studio-freight/lenis) for smooth scrolling
-   **Deployment**: Vercel / Netlify (Ready)

## 📦 Getting Started

Follow these steps to set up the project locally.

### Prerequisites

-   Node.js (v18 or higher recommended)
-   npm or yarn

### Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/AtulBangre/fakhri-next.git
    cd fakhri-next
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server:**

    ```bash
    npm run dev
    ```

4.  **Open your browser:**
    Navigate to [http://localhost:3000](http://localhost:3000) to view the application.

## 📜 Scripts

-   `npm run dev`: Starts the development server.
-   `npm run build`: Builds the application for production.
-   `npm run start`: Runs the production server.
-   `npm run lint`: Runs ESLint to check for code issues.

## 📂 Project Structure

```
│   ├── home/             # Homepage sections
│   ├── layout/           # Header, Footer, etc.
│   └── ui/               # Generic UI elements (Cards, Buttons)
├── data/                 # Static data files (navigation, content)

└── styles/               # Global styles
```
